# ZapisyGrupowe — Frontend

Interfejs użytkownika systemu zapisów na grupy laboratoryjne. Branch `demo` zawiera działający frontend **NIE** połączony z backendem.

**Live:** [zapisy-grupowe.xyz](https://zapisy-grupowe.xyz)

---

## Stack

- Vanilla JavaScript
- HTML5 / CSS3
- Bez frameworków - czysty frontend

## Struktura

```
/
├── index.html      ← punkt wejścia
├── assets/         ← obrazy, ikony
├── pages/          ← podstrony
├── styles/         ← arkusze CSS
└── src/            ← logika JS
```

## Połączenie z backendem

Po implementacje lokalną z backendem, uruchom serwer z brancha `backend-integration` — instrukcja w [`backend/README.md`](../backend/README.md).
